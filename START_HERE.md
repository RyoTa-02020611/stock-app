# 🚀 今すぐ始める：登録なしで使えるアプリをデプロイ

## このアプリの特徴

✅ **既に登録なしで使える状態です！**
- ユーザー登録不要
- ログイン不要
- アカウント作成不要
- データは各ユーザーのブラウザに保存（プライベート）

## 📋 デプロイ手順（3ステップ）

### ステップ1: Vercelにログイン

PowerShellで実行：
```powershell
vercel login
```

ブラウザが自動で開くので：
1. GitHubアカウントでログイン
2. Vercelへのアクセスを許可

### ステップ2: デプロイ

```powershell
cd C:\app
vercel --prod
```

初回は質問されます：
- **Set up and deploy?** → `Y` を入力
- **Which scope?** → 自分のアカウントを選択
- **Link to existing project?** → `N` を入力
- **Project name?** → Enter（デフォルト名でOK）
- **Directory?** → `./` を入力（そのままEnterでもOK）

### ステップ3: URLをコピー

デプロイが完了すると、以下のようなURLが表示されます：
```
✅ Production: https://stock-portfolio-xxxxx.vercel.app
```

**このURLをコピーして、メールで共有してください！**

## 🆘 エラーが出た場合

### ログインできない場合

- ブラウザが開かない場合は、手動で https://vercel.com にアクセスしてログイン
- その後、`vercel login` を再実行

### デプロイが失敗する場合

```powershell
# ローカルでビルドをテスト
npm run build
```

エラーがあれば修正してから再デプロイ。
